# Movie Booking Website 🎬

A Django-based web application that allows users to browse movies, view schedules, select seats, and confirm bookings.

## Features
- Movie listing
- Schedule selection
- Seat booking system
- Booking confirmation

## Tech Stack
- Python (Django)
- HTML, CSS
- SQLite

## Setup Instructions
1. Clone the repository
2. Create a virtual environment:
   python -m venv venv
3. Activate it:
   venv\Scripts\activate
4. Install dependencies:
   pip install -r requirements.txt
5. Run server:
   python manage.py runserver
